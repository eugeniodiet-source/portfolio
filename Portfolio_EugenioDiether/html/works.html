Parking Lot Management System

The Puregold Paso De Blas Parking Lot Management System is an application that stores and manages the parking space availability. Karaoke Queuing System

Route Finder

This project demonstrates the application of graph theory in route finding by implementing Dijkstra’s shortest path algorithm along with Yen’s Algorithm. In this project, streets and intersections will be structured as nodes and edges in a graph. The data of nodes and edges are collected in the OverPass Turbo site and compiled to make an Osm File. The system utilized 2 different libraries which are OsmSharp and GMap.Net. OsmSharp is an OpenStreetMap library that extracts data from an osm file. Gmap.Net library helps for visual representation of the map and routes.

The system enables users to enter a starting point and a destination, then the system will compute and display three shortest possible routes based on distance. Each route will be shown on the map using different colors, along with a legend to help the user easily distinguish which path corresponds to which distance.

Cisco Activities

Highlighted here are some cisco networking done by me.

Switch vs Hub

In this activity, I configure similar computer connection with use of switch and hub. The switch only sends data to intended receiver while the hub broadcast the data to all connected clients.

Static Routing

Static routing configuration is manually configuring the path of network traffic to reach its destination. In this configuration, I manually input the destination network, subnet mask, and next-hop IP address of the router.

DHCP Server Configuration

In DHCP (Dynamic Host Configuration Protocol) Server Configuration, I configure a server to give an automatic IP address to each client connected to it. In this way, we won't need to manually configure each PC (Static IP Address) to have connection to one another.

Wireless Routing

The routers aside from using cables and serial DTE, they can also connect clients through wireless connection. In this configuration, one router act as main source of network and the other two routers extends this connection by allowing its client to connect wirelessly.
